# 🔒 基于机器学习的校园网络流量异常攻击检测

> 🚀 北邮计算机学院 · 数据科学与大数据技术专业 · 大一实战项目
> 
> 本项目记录了从“数据作弊满分（v1.0）”到“特征高度混淆翻车（v2.0）”，再到最终通过“验证集阈值调优与三数据集划分”完成“极限挽救（v3.0）”的完整网安大数据博弈历程。

## 📁 仓储文件指南
* 📜 `network_security_v3.py`: 包含三数据集划分（6:2:2）、StandardScaler预处理管道、以及 0.01~0.99 自动化阈值搜索的完整可运行代码。
* 💡 `network_security_anomaly_detection.md`: **精美研究笔记 / 电子错题本**。详尽记录了不平衡网安场景下的业务常识、特征重叠尸检反思、以及树模型对标准化免疫等硬核考点。
* 📈 `f1_score_comparison.png`: 三模型 F1-Score 对比柱状图（最优阈值 vs 默认阈值）。
* 📉 `pr_curve_xgboost.png`: XGBoost 模型的 Precision-Recall 权衡曲线，直观展示“宁可错杀、绝不放过”的业务博弈。

## 🏆 阶段性实战成果（v3.0 最优阈值战报）
面对正常与攻击特征高度重叠的极难数据集，通过将标准阈值从 0.5 调优降至 0.15~0.21 区间，成功实现挽救：
* **LightGBM (最优阈值 0.17)**: F1-Score 从 0.14 提升至 **0.3545**，捕获黑客的召回率（Recall）暴力飙升 **6.2 倍**（达到 55.30%）。
* **RandomForest (最优阈值 0.15)**: 斩获 **0.6075** 的全场最高召回率，用工程误报代价换取了极高的网络安全性。
